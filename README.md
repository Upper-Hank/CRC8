# CRC8 MAXIM 校验码计算器

专为CRC8 MAXIM算法（多项式0x8C）设计的网页工具，界面简洁、功能专业。

## 🌟 功能特性

### 核心功能
- **CRC8 MAXIM算法**：专用多项式0x8C实现
- **### 测试用例
```
十六进制: C8 64 54 00 00
CRC8-MAXIM: 待测试

十六进制: 01 64 FF E2 00 00 00 00 11
CRC8-MAXIM: 待测试

文本: "Hello World"
CRC8-MAXIM: 0x07

十六进制: 48656C6C6F20576F726C64
CRC8-MAXIM: 0x07
```

*注：具体的CRC值需要根据您的实际数据进行验证*供的C代码算法逻辑
- **多格式输入**：支持文本字符串和十六进制数据输入
- **多格式输出**：显示十六进制、十进制和二进制结果
- **实时计算**：输入时自动计算（300ms防抖）

### 用户体验
- **现代化界面**：采用现代化设计语言，支持深色/浅色主题切换
- **响应式设计**：完美适配桌面和移动设备
- **历史记录**：自动保存最近10次计算记录
- **快捷操作**：一键复制结果、快速清空输入、示例数据加载
- **时间统计**：显示计算耗时和数据处理信息

### 交互功能
- **键盘快捷键**：支持常用快捷键操作
- **错误处理**：友好的错误提示和输入验证
- **本地存储**：历史记录和主题设置本地保存
- **复制功能**：支持一键复制各种格式的结果

## 📖 使用说明

### 基本操作
1. **选择输入格式**：文本字符串或十六进制
2. **输入数据**：在文本框中输入要计算的数据
3. **选择算法**：从预设算法中选择或使用自定义多项式
4. **查看结果**：系统会自动计算并显示结果

### 输入格式说明
- **文本字符串**：直接输入普通文本，如 "Hello World"
- **十六进制**：输入十六进制数据，支持以下格式：
  - `48656C6C6F` （连续格式）
  - `48 65 6C 6C 6F` （空格分隔）
  - `0x48 0x65 0x6C 0x6C 0x6F` （0x前缀）

### CRC8 MAXIM算法规格
| 参数 | 值 | 说明 |
|------|-----|------|
| 多项式 | 0x8C | 根据您的通信协议规范 |
| 初始值 | 0x00 | CRC计算起始值 |
| 算法实现 | 右移位 | 每位检测最低位，符合C代码逻辑 |
| 应用场景 | 专用通信协议 | 根据您的协议文档实现 |

### 键盘快捷键
- `Ctrl/Cmd + Enter`：快速计算
- `Ctrl/Cmd + L`：清空输入
- `Ctrl/Cmd + C`：复制结果（焦点在结果区域时）
- `F1`：加载示例数据
